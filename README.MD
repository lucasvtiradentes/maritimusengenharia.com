This project was developed when I was working in maritimus engenharia, in 2020, when I notice we needed a website and also a validation system for our unique and advanced Excel spreadsheets.

## :information_source: Features

- [x] Mobile first orientation development;
- [x] Responsivity driven development;
- [x] Validation of company's Excel spreadsheat's by checking user pc/username at a private mongodb database;
- [x] Dynamic update of company's operation numbers by a JSON file;
- [x] Contact form to get in touch, wihch sends information to an email inbox.

## :information_source: How to edit it

To clone this repository and make changes in the source code, you'll need [Git](https://git-scm.com) and  [Nodejs](https://nodejs.org/en/).

```bash
# Clone this repository
$ git clone https://github.com/lucasvtiradentes/maritimusengenharia.com

# Go into the repository
$ cd maritimusengenharia.com

# Install dependencies
$ npm install

# Run the code in devlopment mode
$ npm run dev
```
